## `paletteer` package

This a one stop destination for anyone looking for a color palette to use in `r`.
Following packages are included:
 - `awtools`
 - `cartography`
 - `dichromat`
 - `dutchmasters`
 - `ggsci`
 - `ggpomological`
 - `ggthemes`
 - `ghibli`
 - `grDevices`
 - `jcolors`
 - `LaCroixColoR`
 - `NineteenEightyR`
 - `nord`
 - `ochRe`
 - `oompaBase`
 - `palettetown`
 - `palr`
 - `pals`
 - `Polychrome`
